onCreate() : 앱이 시작하게 되면 가장 먼저 호출되는 함수
액티비티가 생성되는 함수로써 이 함수는 최초 1번만 호출
가장 많이 사용

onStart() : 화면이 사용자에게 보여지기 시작
액티비티가 다른 액티비티에 가려져 있다가 다시 나타날 때 호출되는 함수
다른 액티비티에서 데이터를 받을 때 혹은 화면에 데이터를 Setting 할 때 이 함수에서 해줌

onResume() : 화면이 전면에 드라나 사용자에게 보여지고 실제로 액티비티가 동작하며 호출되는 함수

onPause() : 액티비티가 다른 액티비티에 가려지면 호출되는 함수
일시정지 상태. 데이터를 저장해야 한다면, 이 함수에서 저장

그리고 저장된 데이터를 onStart()에서 가져오면 됨

onStop() : onPause() 상태에서 오랜시간 전면에 드라나지 않으면 호출됨
이때 이제 액티비티가 메모리에서 제거될 준비를 하게 됨

onRestart() : 액티비티가 메모리에서 제거 될 준비를 하고 있는데, 사용자가 그 액티비티를 다시 보려고 하면 호출되는 함수.
이 함수는 onStart()를 부르며 완전히 전면에 드러나게 됨

onDestroy() : 액티비티가 완전히 메모리에서 제거 됨
흔히 앱을 종료하거나, 다른 앱이 실행되어 메모리 용량이 부족해지면 제거되어 호출
